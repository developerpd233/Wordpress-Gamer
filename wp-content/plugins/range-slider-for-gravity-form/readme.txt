== Description ==
**Gravity Forms Range Slider** is a easy to use setup field range in your Gravity Form for your [WordPress](https://en.wikipedia.org/wiki/WordPress) site.

**[Demo](http://oceanwebguru.com/wordpresstest/range-slider-gravity-form/)**

> **Plugin Features**
>
> * Multiple Tooltip Position
> * Step Adding Option
> * Min and Max Option in slider
> * Prefix On Digit
> * Tooltip color option
> * Compatible in all major browser
> * Easy use
